# api-test-petstore
API tests for Petstore web service


# How to run
```java
mvn clean test
```

or 

```java
./mvnw clean test
```

# How to generate and run HTML report

```java
mvn allure:serve 
```
or 

```java
./mvnw allure:serve 
```
